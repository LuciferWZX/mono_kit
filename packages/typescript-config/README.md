# 📦 `@mono-kit/typescript-config`

工作区共享的 TypeScript 配置。

## 📝 简介

这是一个用于统一项目 TypeScript 配置的共享包。它提供了一致的类型检查和编译选项，确保所有子项目使用相同的 TypeScript 规范。

## ✨ 特点

- 统一的 TypeScript 配置方案
- 共享的类型定义文件
- 标准化的编译选项

## 📥 使用方法

1. 在项目中安装依赖:

```shell
    pnpm --filter <target-package-name> add -D @mono-kit/typescript-config
```
